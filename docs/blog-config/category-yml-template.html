_category_.yml 模板（完整注释样例）

以下与仓库根目录 config-examples/_category_.yml 内容一致；在站内阅读时可直接复制下面代码块。将文件保存为 _category_.yml 并放到目标文档目录下即可。

各字段含义的展开说明见 分类元数据说明。

# =============================================================================
# Docusaurus 侧栏「分类」元数据 — 样例（可复制到 docs/<某层目录>/ 下并重命名为 _category_.yml）
# =============================================================================
# 官方支持 _category_.json 与 _category_.yml，字段等价；本仓库 docs/ 已统一使用 .yml（可写 # 注释）。
# 顶栏分区名称：docs-nav-autogen 读取「一级目录」下本文件的 label；若仍存在 .json（少见），优先 .yml。
# 文档：https://docusaurus.io/docs/sidebar/autogenerated#category-metadata
# =============================================================================

# -----------------------------------------------------------------------------
# label（可选，字符串）
# -----------------------------------------------------------------------------
# 侧栏里这一层分类的显示名；一级目录时在本站还会用作顶栏该分区的标签。
# 不写则由文件夹名推导（如 my-topic → My Topic）。
label: 示例分区

# -----------------------------------------------------------------------------
# position（可选，数字）
# -----------------------------------------------------------------------------
# 同一父级下，分类排序；越小越靠前，可用小数插在两项之间。
# position: 1

# -----------------------------------------------------------------------------
# collapsed（可选，布尔）
# -----------------------------------------------------------------------------
# 首次打开站点时，该分类在侧栏默认收起 (true) 还是展开 (false)。
collapsed: false

# -----------------------------------------------------------------------------
# collapsible（可选，布尔）
# -----------------------------------------------------------------------------
# 该分类在侧栏是否允许折叠（是否显示箭头）。默认 true；false 则始终展开且不可折叠。
# collapsible: true

# -----------------------------------------------------------------------------
# className（可选，字符串）
# -----------------------------------------------------------------------------
# 加到该分类 DOM 上的 CSS 类名，便于自定义样式。
# className: my-category

# -----------------------------------------------------------------------------
# link（可选，对象）— 点击侧栏「分类标题」时的行为；结构以当前 Docusaurus 版本为准
# -----------------------------------------------------------------------------
# 常见：generated-index 自动生成索引页，或指向某篇文档。
# link:
#   type: generated-index
#   title: 概览标题
#   description: 简短描述
#   slug: /overview

# -----------------------------------------------------------------------------
# customProps（可选，对象）
# -----------------------------------------------------------------------------
# 挂在分类上的自定义数据，供主题或 swizzle 组件使用。
# customProps:
#   badge: 新
