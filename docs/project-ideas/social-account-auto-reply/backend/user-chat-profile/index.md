# 用户消息流水与画像

## 消息流水

- 扩展上报「某用户发来新消息」时（见 [新消息上行](../../client-extension/inbound-messages/index.md)），按 **平台 + 对方用户标识**（及会话 id）**追加写入**聊天记录：时间、方向、文本或摘要等，便于审计与回放。

## 用户画像

- 在流水之上 **异步或按需** 生成 / 更新画像（兴趣、意图、沟通风格、禁忌等），供系统提示词或独立读取逻辑使用。  
- **回答策略**可显式依赖「画像 + 近期对话」，而非仅单条消息。

## 对齐与合规

- 上报字段与扩展侧 [新消息上行](../../client-extension/inbound-messages/index.md) 对齐；画像更新频率、隐私与合规在落地产品中单独约束。
