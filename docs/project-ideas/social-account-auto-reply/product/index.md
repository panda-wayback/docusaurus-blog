# 产品与变现

## 定位

主要服务在**抖音、快手、小红书**等平台经营账号的个人或团队：为**私信 / 评论等消息**提供自动回复能力（可配置全自动、半自动或仅生成草稿）。技术栈与 [架构与数据流](../architecture/index.md) 一致。

## 变现设想（可组合）

- **订阅制 SaaS**：按绑定账号数、消息量、并发会话等分档月付 / 年付；记忆体与推理托管在你方。  
- **增值模块**：按平台的「站点适配包」、人设模板、团队审核流等加价。  
- **落地服务**：话术与记忆体初始化、运维托管，收实施费或年费。  

定价与核心客群（达人 MCN、中小商家、客服外包等）需在需求验证后再定。
